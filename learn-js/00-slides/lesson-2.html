Functions

Creating and calling a simple function

var divideByThree = function(num) {
    var val = num / 3;
    console.log(val);
};
 Create a variable and assign it a function
Create a variable called val
assign it the value of number / 3
Output val to the console 
divideByThree(6);
 we call the function 'dividebythree'
and input the parameter 6
The computer then runs the code inside the function!
